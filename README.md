# SWTestApp

## Описание

Необходимо создать приложение, которое будет представлять из себя галерею изображений с двумя экранами **List** и **Detail**.

## Используемые технологии

<li>Swift 5</li>
<li>iOS 14+</li>
<li>SwiftUI</li>
<li>Combine</li>
<li>Pagination</li>
<li>MVVM</li>
